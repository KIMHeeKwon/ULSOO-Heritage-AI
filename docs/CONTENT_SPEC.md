## 9. 어조·표기 규칙 (전 섹션 공통)

1. 문체: 본문은 `~합니다`체. 명사형 종결(`~구축`, `~달성`)은 하이라이트·표 안에서만.
2. 숫자: 4자리 이상은 콤마(31,022). 단위는 붙여 쓴다(200억 원 → 화면에서는 `200억+`처럼 축약 가능).
3. 고유명사 표기 고정: `국립중앙박물관` `국가유산청` `문화체육관광부` `한국콘텐츠진흥원` `TTAK.KO-10.1621` `CIDOC-CRM` `ICCV 2025` `COLING 2022` `AVICOM 2025`
4. 금지 표현: `CVPR 게재`(제출 단계), `SCI 저널 N편`(CHIC 실적 0), `세계 1위`류의 근거 없는 최상급. `세계 최초`는 4계층 데이터 패브릭에 한해서만, `국내 최초`는 생성 표준·취득 가이드라인에 한해서만 사용.
5. ULSOO 과제 성능 수치(SSIM 95%, 100단계, 0.5µm, ΔE≤2.0)는 반드시 `목표` 또는 `개발 중` 문맥과 함께.
6. 영문 병기는 처음 1회만: `검색증강생성(RAG)` 이후에는 `RAG`.
